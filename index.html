Integral de Riemann

uma ferramenta educacional

View project onGitHub

Este recurso educacional aberto (REA) é uma ferramenta que permite ao educador e ao aprendiz explorar os seguintes conceitos de Cálculo Integral:

  • Soma de Riemann
  • Integral de Riemann
  • Teorema fundamental do cálculo
  • Teorema do valor médio para integrais

Quero usar

No momento há duas versões deste aplicativo: uma para navegadores (aqui) e outra para desktop (aqui).

A única diferença entre eles é que a versão para desktop é capaz de salvar arquivos de atividades, o que lhe permite parar e retomar um trabalho no meio. Na versão para navegadores você também pode fazer isso, mas precisará copiar para um arquivo de texto (.txt) um string gerada pelo aplicativo.

Este aplicativo ainda não tem uma versão para tablets e smartphones. Aguarde, por favor.
